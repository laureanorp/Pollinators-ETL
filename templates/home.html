RFID Pollinators

This pipeline produces useful charts and statistics based on experiments using Biomark hardware and RFID antennas. Signal information is analyzed, processed and categorized based on the user's input parameters. Start by choosing one or more Excel files.

Please choose one or more .CSV files to upload

Straightforward

Define the genotypes for each antenna, specify a few parameters and get the results in seconds.

Versatile

Accepts as many antennas on the experiment as necessary. Accepts rotations of the antennas when loading several experiments.

Fast

Based on Pandas and NumPy, two of the fastest and more efficient libraries for data analysis with Python.

Paper-ready results

Graphs are interactive and can be downloaded; and all processed data can be viewed and downloaded if necessary.
